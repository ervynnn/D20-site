## 
    Class V Sensor Array (PL 7)
   



    As the Class IV sensor array, except that the ship also gains a +2 equipment bonus on all initiative checks. This array is normally installed on ships built for combat.
   



    Purchase DC: 36.
   



    Restriction: None.
