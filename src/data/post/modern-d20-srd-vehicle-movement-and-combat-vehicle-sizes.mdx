##

    Vehicle Sizes

[Vehicles](modern-d20-srd-equipment-equipment-vehicles)
use the same size categories as characters and creatures, as shown on Table: Vehicle Sizes. The
[vehicles](modern-d20-srd-equipment-equipment-vehicles)
size modifier applies to its
[initiative](modern-d20-srd-combat-initiative)
modifier,
[maneuver](modern-d20-srd-vehicle-movement-and-combat-simple-maneuvers)
modifier, and
[Defense](modern-d20-srd-combat-defense)
. (The size modifier is already included in the vehicle statistics on
[Table: Vehicles](modern-d20-srd-equipment-equipment-vehicles)
)

| Table: Vehicle Sizes |     |                              |
| :------------------- | :-: | ---------------------------: |
|                      |     |                              |
| Colossal             |  8  |     Yacht, semi with trailer |
| Gargantuan           |  4  |              Tank, limousine |
| Huge                 |  2  | Luxury car, SUV, armored car |
| Large                |  1  |          Economy car, Harley |
| Medium-size          | +0  |       Racing bike, dirt bike |
