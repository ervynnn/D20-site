##  Gearhead

Benefit: The character gets a +2 bonus on all [ Computer Use ](modern-d20-srd-skills-computer-use) [ checks ](modern-d20-srd-skills-skill-basics-php#skill-check-index) and [ Repair ](modern-d20-srd-skills-repair) [ checks](modern-d20-srd-skills-skill-basics-php#skill-check-index) .

Special: Remember that the [ Computer Use ](modern-d20-srd-skills-computer-use) skill and the [ Repair ](modern-d20-srd-skills-repair) skill can only be
used [ untrained ](modern-d20-srd-skills-skill-basics) in certain situations.
