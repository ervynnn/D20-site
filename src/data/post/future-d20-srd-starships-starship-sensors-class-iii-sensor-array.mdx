## 
     Class III Sensor Array (PL 6)
    



     As the Class II sensor array, except that the ship also gains a +2 equipment bonus on all initiative checks. This array is normally installed on ships built for combat.
    



     Purchase DC: 30.
    



     Restriction: None.
