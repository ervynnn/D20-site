## 
    Class VII Sensor Array (PL 8)
   



    As the Class VI sensor array, except that the ship also gains a +2 equipment bonus on all initiative checks. This array is normally installed on ships built for combat.
   



    Purchase DC: 42.
   



    Restriction: None.
