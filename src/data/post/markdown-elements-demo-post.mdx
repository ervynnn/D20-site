## Webmaster's Note

Dept7 was created to make playing d20 games easier. There are many RPG books available, but having every book on hand can be cumbersome. My goal was to create an online reference for quick rule lookups, reducing downtime during games.

I initially built this site to support my own d20 Modern campaign. Using the OGL, I converted the Modern d20 SRD into a web format with cross-referencing, making it a more efficient resource than physical books. This site does not replace official books but serves as a handy supplement.

I hope to expand this project with more OGL systems to make it a broader resource for gamers. If you know of any open systems that could benefit from this format, please [contact me](contact).

For those of us who have grown up with RPGs and have less free time, this site is for you. May it help you spend less time searching and more time playing.

**- Brendan Quinn**
