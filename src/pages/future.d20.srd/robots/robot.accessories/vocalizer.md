##  Vocalizer (PL 5)

This unit enables a robot to speak any language it knows. It must have the
appropriate Speak Language skill, either acquired through class levels (for
heroic robots) or skill software (for nonheroic robots).

Purchase DC: 10.

Restriction: None.
